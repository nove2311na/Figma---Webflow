# Workspace Artifact Schemas

Runtime artifacts live under `workspace/` and are generated by the pipeline compiler. The schemas in `agentic/schemas/` define the expected shape.

## Core Folders

- `workspace/figma/` — Stable figma node extraction bundles and normalized trees.
- `workspace/semantic/` — Semantic IR trees.
- `workspace/html/` — Generated local HTML files and blueprints.
- `workspace/html/chunks/` — HTML slices for sections.
- `workspace/webflow-native/` — Webflow branch build plans, task logs, and write audit logs.
- `workspace/reports/` — Design-system sync, component matching, and QA verification reports.

## Core Files & Schemas

| File/Path | Producer | Purpose | Schema |
|---|---|---|---|
| `workspace/figma/figma.node-bundle.json` | operator | Persisted Figma raw extraction tree. | `agentic/schemas/figma-node-bundle.schema.json` |
| `workspace/figma/figma.normalized-tree.json` | compiler | Normalized Figma tree. | `agentic/schemas/figma-normalized-tree.schema.json` |
| `workspace/semantic/figma.semantic-tree.json` | compiler | Semantic role/tag IR tree. | `agentic/schemas/figma-semantic-tree.schema.json` |
| `workspace/html/page.blueprint.json` | architect | Logical page layout blueprint. | `agentic/schemas/html-blueprint.schema.json` |
| `workspace/html/page.html` | compiler | Rendered HTML source page. | (Raw HTML Output) |
| `workspace/html/asset-manifest.json` | compiler | Asset role & alt manifest. | `agentic/schemas/asset-manifest.schema.json` |
| `workspace/webflow-native/native-build-plan.json` | compiler | Serialized Webflow mutation plan. | `agentic/schemas/webflow-native-build-plan.schema.json` |
| `workspace/webflow-native/write-audit-log.jsonl` | operator | Transactional audit log of writes. | `agentic/schemas/webflow-write-audit-log.schema.json` |
