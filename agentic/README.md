# Agentic System

This folder stores durable agentic design, rules, policies, memory, schemas, and generated specs.

| Folder | Purpose |
|---|---|
| `specs/` | Seed input, `agent_system_spec`, artifact contracts, and validation report. |
| `schemas/` | JSON schema contracts for generated workspace artifacts. |
| `knowledge/` | Stable project knowledge and source maps. |
| `memory/` | Memory candidates and approved memory cards. |
| `policies/` | Tool risk, approval, no-overwrite, and MCP auth rules. |
| `rules/` | Class selection, HTML QA, concurrency, retry, and Webflow MCP rule policies. |

This folder is the V3 agentic quality layer on top of the existing MAS workflow.
