# Source CSS

This directory contains the original Client-First CSS files used to generate the library contract:
- `normalize.css`: Normalization styling rules.
- `webflow.css`: Webflow native engine reserved classes.
- `client-first-v2-2.webflow.css`: The Finsweet Client-First V2.2 stylesheet containing all design tokens and utility classes.

The generated contract is the source of truth for all compiler operations.
