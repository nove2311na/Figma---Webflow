# Figma to HTML Architect References

Load these on-demand, in this order:

1. **`figma-naming-convention.json`** — Read first when validating layer names. The approved `data-name` conventions and semantic hints for resolving generic Figma HTML into native HTML tags.
2. **`html-semantic-mapping.json`** — Read when mapping raw Figma HTML to final Webflow HTML. The rule objects in this file drive `process_html.py`'s semantic resolution (semantic / layout / interactive / form / typography / media / decorative categories). Do not edit this file to add one-off rules — extend the JSON schema and update `process_html.py` together.

Do not duplicate content from these files into the SKILL.md body — load them on demand. If you add project-specific semantic rules, append a new file (e.g. `semantic-extensions.json`) and link it from here.
