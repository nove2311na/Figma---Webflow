# v0.7.0 HTML-First Compiler Architecture Start

Date: 2026-06-03

## Overview

Initiated the comprehensive refactoring of the Figma-to-Webflow pipeline into a strict HTML-first compiler architecture. This release includes Phase 0 (Archive cleanup), Phase 1 (Folder structure, specs, rules, and schemas), Phase 2 (Client-First CSS library contract compiler), and Phase 3 (Figma MCP extraction contract and schemas).
