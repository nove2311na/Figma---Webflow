## Key Corrections Applied (the big ones)

1. **CF V2.2 ships layout utilities** — flex-horizontal/vertical, gap-*, grid-N-col, grid-autofit/autofill.
   Previous rule "CF has ZERO layout utilities, always Case 1" was WRONG. Deleted from all files.

2. **Spacing is variable-backed + auto-responsive** — Layout collection modes handle tablet/mobile spacing shifts.
   Previous guidance "compute px/16, round to clean rem" was wrong. Now: token-snap + apply utility.

3. **aspect-ratio-portrait = 2/3 not 3/4; landscape = 3/2 not 4/3.** All aspect-ratio classes include object-fit:cover.

4. **text-weight-medium EXISTS.** Auditor was wrong. V2.2 ships 6 weights.

5. **text-size-huge does NOT exist.** Removed from all catalogs.

6. **Case 4 (breakpoint custom class) now applies to structural reflow only.** Spacing/type handled by variable modes.

7. **`_component` wrapper suffix.** Component outermost wrapper = `name_component`; children = `name_el`.

8. **Button combos: only `is-secondary` and `is-text`.** Removed `is-brand`/`is-outline` examples.
