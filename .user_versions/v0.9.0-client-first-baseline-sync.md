# v0.9.0 Figma-to-Webflow MAS Upgrade & Client-First Baseline Sync

Date: 2026-06-04

## Overview

Upgraded the Figma-to-Webflow Multi-Agent System (MAS) skills (`[new]-design-system-sync` and `[new]-figma-to-html-architect`) from scaffolds to production-ready scripts. Integrated the Client-First Baseline Addendum: isolated the Finsweet Client-First CSS layer from Webflow native/core CSS, preventing Webflow native classes (`.w-*`, `.wf-*`) and HTML tag rules from bleeding into the Figma design system or HTML compilation. Archived deprecated skills folders to keep the workspace clean.
